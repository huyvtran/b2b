Replace cordova on below location:
cisco_b2b\platforms\android\CordovaLib\src\org\apache

Need To install MVF plugin:
	Go to terminal 
	Go to path till your root directory i.e. cisco_b2b
	Run this command :
	cordova plugin add Releases/Plugin/MVFPlugin

Need To Auth_plugin:
	cordova plugin add Releases/Plugin/cisco-plugin-auth



Other Plugin - Must to Install 
	ionic plugin add cordova-plugin-inappbrowser
	cordova plugin add cordova-plugin-sslcertificatechecker
	cordova plugin add cordova-plugin-network-information
